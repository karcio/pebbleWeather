# Pebble Weather
================
![](https://github.com/karcio/pebbleWeather/blob/master/resources/images/weather_screenshot.png)

Simple information about weather in selected city:
- city name
- degrees in Celsius
- sky state
- humidity
- pressure
- wind speed
- wind direction
